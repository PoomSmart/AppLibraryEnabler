# App Library Enabler
Turn on App Library

## Compatibility
iOS 14 and above

## Features:
* enables App Library on iPadOS 14
* add toggles to control App Library settings under "Home Screen & Dock" - just like you can on iOS 14 natively

**No options to configure**
